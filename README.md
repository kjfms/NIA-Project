## 1. 분석 개요
- #### 2020년 장흥군청 정보공개청구 민원 데이터를 활용해 민원 신청의 현황, 추이 등을 다양한 방면에서 분석 진행
  - 정보공개 신청 현황을 여러 유형별(월별, 요일별, 부서별 등)로 분석 진행
  - 전체 주요 민원 키워드 및 부서별 주요 민원 키워드의 현황 및 추이 분석 진행
